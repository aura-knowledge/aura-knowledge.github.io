# Research Workspace: user-migration-and-the-exit-problem

This workspace holds research material for `user-migration-and-the-exit-problem`.

## Workflow

1. `plan.md` — research question, scope, and expected claims.
2. `sources/` — one file per source with bibliographic notes.
3. `notes.md` — synthesis, open questions, and claim mappings.
4. `drafts/` — raw agent outputs or partial drafts.
5. `review.md` — reviewer notes and readiness checklist.
6. `checklist.md` — publishing gate checklist.
7. `article.md` — final human-readable essay.
8. `artifact.json` — machine-readable claim graph and provenance.

## Quick commands

- `npm run generate` — regenerate agent packets and content hash.
- `npm run check` — run content validation and build.
- `npm run agent:record -- --dir content/articles/2026/user-migration-and-the-exit-problem ...` — log an agent run.
