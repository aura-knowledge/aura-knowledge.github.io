## Summary

ActivityPub is a W3C recommendation for a decentralized social-networking protocol. It defines how servers exchange activities such as follows, posts, likes, and updates, enabling interoperability between independent social platforms.

## Relevance to claims

- claim-004: Directly supports the claim that federated protocols allow users to change providers without leaving their networks; the protocol is the shared language between servers.

## Key snippet

> The ActivityPub protocol is a decentralized social networking protocol based upon the ActivityStreams 2.0 data format. It provides a client to server API for creating, updating and deleting content, as well as a federated server to server API for delivering notifications and content.
